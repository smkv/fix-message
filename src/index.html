FIX message decoder

Enter a FIX message to decode. All processing is done locally in your browser, so your data remains secure and is never sent to a server. The standard SOH delimiter is always supported. E.g. 8=FIX.4.2|9=65|35=A|...|10=059|
Powered by fix-message, an open-source Web Component for decoding FIX messages.
